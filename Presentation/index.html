Le Concept

  • Asymétrie : Attaquant vs Défenseur
  • Temps Réel : Synchronisation SignalR
  • Tactique : Gestion de ressources et placement stratégique

Architecture Technique

Clean Architecture & .NET 9

Domain : Simulation pure, sans dépendances.

Application : Gestion des sessions et des commandes.

UI.Desktop : Avalonia UI + SukiUI (Modern Desktop).

Infrastructure : Client SignalR (MessagePack).

Server : Hub ASP.NET Core & Groupes.

Utilisation de SignalR Groups


// Isolation des joueurs dans des salles de combat
await Groups.AddToGroupAsync(connectionId, gameId);

// Diffusion des actions uniquement au groupe concerné
await Clients.Group(gameId).ReceivePlayerAction(action);

Protocole MessagePack pour une latence minimale.

Simulation Déterministe

  • Le serveur génère une Seed unique au lancement.
  • La simulation tourne à 60 Ticks/seconde.
  • Chaque client calcule sa propre frame : Désynchronisation impossible.
